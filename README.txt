Запуск через Docker:

1) В корневой папке игры открыть терминал либо через терминал зайти в корневую папку KosinkaPHP
2) В терминале создаем образ приложения "docker build -t kosinka ."
3) После там же в терминале создаем контейнер и сразу же запускаем его "docker run -it --name kosinka kosinka" 
4) Для повторного запуска контейнера "docker start -i kosinka"
